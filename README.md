# demo-drizzle

## features

- vite
- react
- typescript
- drizzle
- sqlite
- express

## usage

1. install

client:

```bash
pnpm install
```

server:

```bash
pnpm install
cp .env.example .env
# set env
pnpm db:push
pnpm db:init
```

2. run

client:

```bash
pnpm d
pnpm b
pnpm p
```

server:

```bash
pnpm d
pnpm b
pnpm p
```
